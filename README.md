# FinDoc QA

Financial Document Question Answering powered by Retrieval Augmented Generation (RAG).

### Features

- Multi-document financial QA
- PDF upload and indexing
- MiniLM embeddings
- FAISS vector search
- Llama 3.1 via Groq
- Confidence scores
- Page-level citations

Built with:

- Streamlit
- SentenceTransformers
- FAISS
- Groq API
